## Getting Started

To install the application locally you will need:

* PHP >= 7.1.3
* git
* composer

### Installation

1. Clone the repo
```sh
git clone https://github.com/StBlackJesus/SerwuSan.git
```
2. Install dependencies
```sh
composer install
```
